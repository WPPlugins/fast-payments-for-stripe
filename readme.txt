Process credit cards via stripe. Fast setup. Easy to use. Mobile friendly.

== Description ==

Start processing credit cards right from your blog with WP Fast Payments. No web design experience needed! No programming experience needed. Simply configure the plugin and use highly optimized checkout forms generated by Stripe.

WP Fast Payments lives up to its name. It's lightning fast and doesn't require the user to leave your blog.

This plugin does not work with 3rd party e-commerce systems.

= FEATURES =
* Live/Test Mode - Allows you to test the checkout process before going live.
* Set a business/store name.
* Process payments in different currencies.
* Redirect customers to a specific page after a successful payment.
* Redirect customers to a specific page after a failed payment.
